## Run locally

1. Clone repository

```bash
git clone https://github.com/ssynowiec/lucia-demo.git
cd lucia-demo
```

2. Set up environments

```dotenv
DATABASE_URL=
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
```

3. Install dependencies & run project

```bash
npm install
npm run dev
# or
yarn install
yarn dev
# or
pnpm install
pnpm dev
# or
bun install
bun dev
```
